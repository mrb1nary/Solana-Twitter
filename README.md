# 🐦 Solana Twitter Clone

Welcome to the Solana Twitter Clone! This project is a decentralized Twitter-like application built on the Solana blockchain. It features a backend implemented with Anchor and Rust, and a frontend developed using React and Chakra UI.

## 🚀 Features

- **Tweet Creation**: Post and manage tweets directly on the Solana blockchain.
- **Wallet Integration**: Sign transactions and interact with the blockchain using Solana Wallet Adapter.
- **Real-Time Updates**: Fetch and display tweets in real-time.
- **Responsive UI**: Enjoy a modern and responsive user interface built with Chakra UI.

## 🛠️ Technologies Used

- **Solana**: Blockchain platform for decentralized applications.
- **Anchor**: Framework for building Solana smart contracts.
- **Rust**: Programming language used for writing smart contracts.
- **React**: JavaScript library for building the frontend.
- **Chakra UI**: Component library for building the user interface.
- **Tailwind CSS**: Utility-first CSS framework for styling.

## 💻 Setup and Installation
### Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or later)
- [Solana CLI](https://docs.solana.com/cli/install-solana-cli-tools)
- [Anchor CLI](https://project-serum.github.io/anchor/getting-started/installation.html)
- [Rust](https://www.rust-lang.org/tools/install)

### Backend Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/solana-twitter-clone.git
    ```
2. Navigate to the `backend` directory:
    ```bash
    cd solana-twitter-clone/backend
    ```
3. Install dependencies and build the project:
    ```bash
    anchor build
    ```

### Frontend Setup

1. Navigate to the `frontend` directory:
    ```bash
    cd solana-twitter-clone/frontend
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```

## 🔧 Configuration

- **Solana Network**: Make sure you are connected to the appropriate Solana network (mainnet, testnet, or local network).
- **Wallet Configuration**: Ensure you have a Solana wallet set up and connected.

## 📄 Usage

1. **Create a Tweet**: Use the frontend to compose and post tweets.
2. **View Tweets**: Browse tweets posted by other users.
3. **Delete Tweets**: Remove your own tweets from the blockchain.
